Tic Tac Toe Python Game

The purpose of this program is to implement a multiplayer (local) game of Tic Tac Toe.

It functions by taking alternating turns, asking the player where they want to play on thier turn, positions (1-9)

Step 1

The first step to implementing this Tic Tac Toe Game, was to create the board so that you can play the game and input where you want to play. I did this by creating a 2d array as a 3x3 matrix.

I then had to add a way to draw the board so that it would look like the game we all know, which I took care of using this function.

Step 2

The next step is to understand where the player wishes to play when it is their turn. This was taken care of by this function get_loc(). This function promts the user where they wish to play and saves saves that value for later use. Also handing edge cases.

Note: this function uses 2 varaibles which are initialized in the begining of the code. We have turn_counter = 0 to keep track of how many turns are played. Also numbers which is a list that keeps track of the positions on the board that have already been played on. The game decides on whos turn it is "X" or "O" based on the turn_count.

Now that we have found where the user wants to play, and updated the board, we must now call draw_board() again so that the user can see the updated game board!

Step 3

One of the last steps within the game are checking if the player has won! For this I used 3 functions.

For the horizontal and verital checks they are a little more complicated than the diagonal. For these two we loop through each row/column searching if we have a winner. We use the match() function to check if all 3 in the row/column are the same as seen below. As for the diagonal positions, there is only 2 possible scenarios so we can check them directly.

Step 4

The fourth and Final step puts it all together using the run_game() function and our main() function

As You see the run_game() function puts all of the previous functions to use, using a loop for each round the game is played. Again the game heavily relies on the turn_counter as the winner is decided based on what turn it is during at the end of the game.

The last important piece to this game is the reset function. This function effectively resets the game so that you can play the game multiple times without having to restart the program.
